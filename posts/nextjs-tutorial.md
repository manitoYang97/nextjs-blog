# Next.js 入门教程

Next.js 是一个基于 React 的全栈框架，它提供了许多强大的功能来帮助你构建现代 Web 应用。

## 为什么选择 Next.js？

1. **服务端渲染 (SSR)** - 提高首屏加载速度和 SEO
2. **静态站点生成 (SSG)** - 构建时生成静态页面，部署简单
3. **文件系统路由** - 基于文件的路由系统，直观易用
4. **API 路由** - 内置 API 端点支持
5. **图像优化** - 自动优化和提供响应式图像

## 快速开始

```bash
npx create-next-app@latest my-app
```

## 项目结构

```
my-app/
├── app/           # App Router
├── public/        # 静态资源
├── components/    # 组件
└── next.config.js # 配置文件
```

## 总结

Next.js 是一个非常强大的框架，适合构建各种类型的 Web 应用。无论是个人博客还是企业级应用，Next.js 都能胜任。
